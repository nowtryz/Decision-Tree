# Decision Tree algorithms (ID3 / C4.5)

This is a python implementation of ID3 and a part of C4.5 algorithms from Ross Quinlan.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install dependencies.

```bash
pip install -r ./requirments.txt
```

## Usage

```python
python -m main.py
```

## Running tests
Some tests are present to ensure accuracy and make sure that the implementation of a new bottlenecked gain formula does not break the rest of the program.

to run them, just use:
```bash
pytest
```
